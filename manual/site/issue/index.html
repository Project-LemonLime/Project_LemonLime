    常见问题及回答

    在使用中出现任何问题，可以在 Github 仓库 中的 Issue 提出。

    此节内容长期更新，更多问题欢迎提出。

    测评时使用更多栈空间

    Windows 平台下可以在 g++ 编译时添加 -Wl,--stack=2147483647 命令来开启约 2048 MB 栈空间。

    Linux 平台下的栈空间限制和题目的内存限制相同。

    第一个点 TLE
